# markben

#### Description
** markben ** 是一套企事业办公管理系统；
主要模块（功能）有：组织机构、用户信息，角色管理、权限配置等基础信息的管理；表单管理（表单设计器），流程管理（流程设计器）、报表管理（报表设计器）等；
最终实现的目标是一套完全的SaaS化企业管理系统（集成钉钉及企业微信）。
